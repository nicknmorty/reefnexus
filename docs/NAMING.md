# NAMING.md — ReefNexus naming decisions

## Adopted split

- **ReefNexus** — project/repo name (architecture and coordination hub).
- **ReefRelay** — core coordination skill/module (handoff/routing/contracts).
- **Clawstro** — lobster-conductor mascot/brand flourish.

## Why this works

- **ReefNexus** signals a connected ecosystem and control center for many specialist agents.
- **ReefRelay** communicates practical coordination behavior (handoff, routing, delivery).
- **Clawstro** keeps the lobster personality and orchestration identity memorable.

## Naming criteria

- Must fit multiagent architecture and coordination, not generic SaaS.
- Must remain understandable in technical docs.
- Should be memorable and brandable in conversation.
- Should allow module naming without ambiguity.
